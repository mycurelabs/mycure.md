### Install dependencies

```bash
yarn
```

### Run development

The `.env.development` file is required to start the development. Refer to `.env.example`.

Run

```bash
yarn serve
```

### Build production

The `.env.production` file is required to start the development. Refer to `.env.example`.

Run

```bash
yarn build
```

### Deploy

Run

```bash
yarn deploy
```
